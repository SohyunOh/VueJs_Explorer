[ Vue 인스터스 ]

1. 데이터 객체

var data = { a:1 }

2.Vue 인스터스에 데이터 객체를 추가합니다.

var vm = new Vue({

data : data

})

3. 인스턴스에 있는 속성은 원본 데이터에 있는 값을 반환합니다

vm.a === data.a // => true

4.인스턴스에 있는 속성 값을 변경하면

//원본 데이터에도 영향을 미칩니다.

vm.a = 2

data.a // =>2

5. 반대의 경우도 영향을 미칩니다.

data.a = 3

vm.a // => 3


* data에 있는 속성들은 인스턴스가 생성될 때 존재한 것들만 반응형
